# **App Name**: SyncStream

## Core Features:

- Screen Sharing: Enable secure screen sharing with low latency for synchronized viewing.
- Video Sync: Ensure all participants view content at the same time.
- Real-time Chat: Real-time chat for text-based communication during watch parties.
- AI Recommendations: Suggest movies and shows based on shared preferences using generative AI.
- Session Management: Allow users to create, join, and manage watch party sessions.
- Emoji Reactions: Add emoji-based reactions to allow real-time, non-verbal responses during the stream.

## Style Guidelines:

- Primary color: Indigo (#4B0082), evoking sophistication and focus.
- Background color: Very dark desaturated purple (#141218) to enhance contrast.
- Accent color: A vibrant magenta (#FF00FF) to highlight interactive elements and call to action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif, providing a modern and neutral aesthetic, making the UI clean and accessible.
- Use clean and consistent icons for easy navigation and enhanced user experience.
- Design a clutter-free layout, focusing on the screen sharing and chat features for a streamlined experience.
- Incorporate subtle animations to provide visual feedback for user interactions and content loading.
